# Basic Ionic core components
## Create/Run/Debug project
### Create project
```
$ ionic start myApp blank|tabs|sidemenu
```
### Run project
```
$ ionic serve
```

### Emulate
```
$ ionic emulate android | ios
```

### Run on device
```
$ ionic run android | ios
```
